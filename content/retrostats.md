# RetroStats: Open-Source Achievement Tracking

**RetroStats** is a Free and Open-Source Software (FOSS) Android application designed for the retro gaming community. It provides a clean, native mobile interface to track your progress, view your unlocked achievements, and browse games on [RetroAchievements.org](https://retroachievements.org/).

### Core Features

* **Native Android UI:** A fast, responsive experience built specifically for Android devices.
* **Official API Integration:** Authenticate safely using your RetroAchievements username and Web API key. 
* **Track Your Progress:** View your recently played games, recently mastered games, and overall completion stats.
* **Free and Open Source:** The code is completely transparent and open for community contributions.

### Secure and Direct
Your RetroAchievements API key is stored securely on your local device and is used exclusively to communicate directly with the official RetroAchievements servers. Two Secret Weapons does not intercept, store, or log your API keys or gaming data.
